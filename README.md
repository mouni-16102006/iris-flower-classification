# 🌸 Iris Flower Classification - CodeAlpha Internship

## 📌 Project Overview
A machine learning project that classifies Iris flowers into 3 species
(Setosa, Versicolor, Virginica) based on their measurements.
Built as part of the **CodeAlpha Machine Learning Internship**.

---

## 🎯 Objective
- Train a classification model on the Iris dataset
- Predict the species of a flower based on measurements
- Evaluate model accuracy and performance

---

## 🛠️ Technologies Used
- Python 3.x
- Scikit-learn
- Pandas
- Matplotlib
- Seaborn

---

## 📊 Dataset
- **Source:** Built-in Scikit-learn Iris Dataset
- **150 samples** — 50 per species
- **4 Features:** Sepal Length, Sepal Width, Petal Length, Petal Width

---

## 🤖 Model Used
- **Random Forest Classifier**
- 100 Decision Trees
- 80% Training / 20% Testing split

---

## 📈 Results
| Metric | Score |
|--------|-------|
| **Accuracy** | 90% |
| Setosa F1-Score | 1.00 |
| Versicolor F1-Score | 0.86 |
| Virginica F1-Score | 0.84 |
